# Debug Records

Use this directory to store practical measurement-debug knowledge.

Recommended entry format:

- Symptom.
- Observed data and status codes.
- Physical checks.
- Instrument checks.
- DUT hypotheses.
- Resolution.
- Whether the result was reproducible.

Default triage order:

1. Physical connection and shielding.
2. Instrument state, correction, range, integration time, and error queue.
3. Measurement data status codes and quality.
4. DUT damage, leakage, process variation, or material effects.

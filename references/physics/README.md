# Physics References

This directory captures concise semiconductor and electrical knowledge used for planning and analysis.

Initial domains:

1. Basic electrical laws for resistors, capacitors, diodes, and MOS structures.
2. B1500A-specific measurement interpretation.
3. Probe, Guard, Kelvin, shielding, and grounding effects.
4. FeFET, FeCap, HfO2/ZrO2 ferroelectric behavior, and oxide TFT behavior.
5. Reliability and defect physics: BTI, HCI, TDDB, interface traps, leakage, and drift.
6. Data analysis: derivatives, integrals, fitting, hysteresis, dispersion, and sanity checks.

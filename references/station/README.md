# Station References

Station references describe the real experiment environment around the B1500A.

Track:

- GPIB address and controller.
- Installed modules and slot-to-channel mapping.
- Probe station, chuck, shielding, and grounding.
- Cable topology, triax lengths, Guard, Kelvin, ASU, SCUU, and selector paths.
- Pin maps for common DUTs.
- Correction and calibration records.

Use example files for public repositories. Keep private lab configuration local unless approved for sharing.
